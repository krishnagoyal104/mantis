# Mantis
 Mantis is a platform to leverage your position on the Aave protocol.
 * Open a long or a short position.
 * Open a position with 3x leverage.

# Checkout
https://www.mantis.net.in/

## Available Scripts

``` 
# install dependencies
yarn

# local development
npm run start

# production build
npm run build
serve -s build
```
